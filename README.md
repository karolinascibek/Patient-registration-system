# Aplikacja Kalendarz

## Technologie 
* PHP 7.4.7
* Vue 2x
* MySQL
* Bootstrap 4.6.0
* uwmap/xampp

## Instalacja 
- pobieramy projekt z https://github.com/karolinascibek/php-vue-app-calendar.git
- composer update

## baza danych
- php artisan migrate
- w pliku .env ustaw baze danych

## Uruchomienie
- npm run watch 
- php artisan serve
